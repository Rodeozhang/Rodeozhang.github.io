实验一 在Github上用Markdown创建技术博客

一、实验目的

熟悉Github网站
掌握Markdown语法
二、实验任务

注册Github账号(Rodeozhang)
创建Gihub Pages repository
发布第一篇博客(my first block)
学习Markdown(访问zybuluo.com,学习现成的语法规则)
利用Hexo生成博客并部署到Github
三、实验步骤与结果

1.安装 node-v7.7.1-x86.msi ;
2.安装 Git-2.1.2.0-32-bit.exe ;
3.电脑 win+R 输入 cmd 回车，执行 npm install -g cnpm –registry=https://registry.npm.taobao.org命令 ;
4.在命令窗口中直接执行 cnpm install -g hexo-cli ;
5.在cmd命令窗口中，创建MD文件夹，执行语句为：mkdir MD ;
6.打开MD，创建hexo文件夹：执行命令为：cd /d D:\MD,回车，执行：mkdir hexo ;
7.打开hexo文件夹，执行hexo init hexo,执行语句为：cd /d D:\MD\hexo,回车，执行hexo init hexo 语句 ;
8.执行 cnpm install ;
9.执行 hexo server.

四、实验小结
