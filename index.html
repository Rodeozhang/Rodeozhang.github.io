实验二

发表于 2017-06-02

实验二 Git的使用

一、实验目的

1.掌握git使用

2.掌握使用Github作为远程仓库

二、实验任务

1.练习本地git操作

2.用SSH同步本地与Github仓库

三、实验步骤与结果

1.同步本地与Github仓库：

下载Github客户端并安装成功

打开Github客户端并登录

在git目录下创建一个名为poem的项目

将创建好的新的项目同步到Github客户端

在客户端可以看到一个未提交的修改

填写评论并提交修改

点击右上角的sync同步按钮将其提交到Github网站上

在Github网站上可以看到新增的commits，表示已实现同步，重复修改同步即可

2.分支开发与合并：

打开Github客户端，点击左上角的create new branch，填写分支名称comment1

在poem文件中写入一句评论语句并保存

提交修改并同步到Github

在Github客户端打开，点击compare&pull request并提交将该分支合并到master中

在Github网站上可以看到新增的branch，表示已实现同步，重复修改即可

四、实验小结

这样在Github客户端中修改更加的简便，并且可以看到修改的次数，对于随时随地的保存和修改项目有着极大的作用。

实验一 在Github上用Markdown创建技术博客

一、实验目的

熟悉Github网站
掌握Markdown语法

二、实验任务

1.注册Github账号 Rodeozhang
2.创建Gihub Pages repository
3.发布第一篇博客 my first blog
4.学习Markdown 访问zybuluo.com,学习现成的语法规则
5.利用Hexo生成博客并部署到Github

三、实验步骤与结果

创建账号

1.访问github.com网站，创建账户 Rodeozhang

2.建立第一个博客 my- first- blog

3.发布第一篇博客，和HQIZCC互相访问账户

4.学习Markdown 访问zybuluo.com,学习现成的语法规则

安装步骤

1.安装 node-v7.7.1-x86.msi ;
2.安装 Git-2.1.2.0-32-bit.exe ;
3.电脑 win+R 输入 cmd 回车，执行 npm install -g cnpm –registry=https://registry.npm.taobao.org命令 ;
4.在命令窗口中直接执行 cnpm install -g hexo-cli ;
5.在cmd命令窗口中，创建MD文件夹，执行语句为：mkdir MD ;
6.打开MD，创建hexo文件夹：执行命令为：cd /d D:\MD,回车，执行：mkdir hexo ;
7.打开hexo文件夹，执行hexo init hexo,执行语句为：cd /d D:\MD\hexo,回车，执行hexo init hexo 语句 ;
8.执行 cnpm install ;
9.执行 hexo server.

四、实验小结

刚接触Blog，觉得很新鲜，在同学的帮助下完成了博客的创建，对于很多不懂的还得好好学习。
